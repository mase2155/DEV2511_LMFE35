4.DOM / Jquery
    - Sự kiện : 
        + viết trên thẻ html : onclick , onblur , ondbclick , ....
        + viết trên file /file script : 
                        ++ DOM : addListenerEvent('sự_kiện' , hàm_xử_lý()){}
                        ++ Jquery : on('sự_kiện', hàm_xử_lý()){}
    - Thay đổi class :
                + DOM : thẻ_nào.display.tên_thuộc_tính_CSS = giá_trị_thuộc_tính_CSS
                + Jquery : thẻ_nào.css(tên_thuộc_tính_CSS , giá_trị_thuộc_tính_CSS)
    - Thay đổi HTML :
                + DOM : thẻ_nào.innerHTML = giá_trị_HTML_thêm
                + Jquery : thẻ_nào.append(giá_trị_thêm)
    - thay đổi vị trí tương tác với thẻ HTML 
                + DOM : child , parent , siblings
                + Jquery : child() , parent() , siblings() , next() , ...
    - tương tác với các attr khác của thẻ HTML
